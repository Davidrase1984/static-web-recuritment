# Opencode Instructions for Project Context

Always load and reference the PROJECT_CONTEXT.md file when starting a new session to understand the project background without needing re-explanation.

## Project Context File
Location: .opencode/plans/PROJECT_CONTEXT.md
Content: Contains comprehensive project overview, technology stack, structure, key files, setup instructions, and current focus.

## Usage
When Opencode starts a new session, it should automatically read this file to understand:
- What the project is about
- What technologies are used
- How the project is structured
- What files are important
- How to set up and run the project
- What the current development focus is

This eliminates the need to re-explain the project basics in each new session.
