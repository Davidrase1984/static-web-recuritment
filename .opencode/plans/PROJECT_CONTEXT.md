# Static Web Recruitment Project

## Project Overview
This is a static web recruitment application built with Vue.js and Vite. The project consists of a frontend application that displays recruitment information.

## Technology Stack
- Vue.js 3
- Vite (build tool)
- HTML/CSS/JavaScript

## Project Structure
- `/src` - Contains Vue components and application logic
  - `/src/App.vue` - Main Vue component
  - `/src/main.js` - Application entry point
  - `/src/functions/` - Utility functions
- `/api` - Backend API (if applicable)
- `/node_modules` - Dependencies
- `index.html` - Main HTML file
- `package.json` - Project dependencies and scripts
- `vite.config.js` - Vite configuration

## Key Files
- `index.html` - Main HTML template
- `src/main.js` - Vue application initialization
- `src/App.vue` - Main application component
- `src/functions/message.js` - Utility function for messaging

## Development Setup
1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Build for production: `npm run build`

## Current Focus
The application is a static recruitment website that displays job listings and company information. It uses Vue.js for reactivity and component-based architecture.

## Notes for Opencode
- This is a Vue 3 application using the Composition API
- The project uses Vite as the build tool for fast development
- Styling is done with standard CSS/Vue scoped styles
- API endpoints (if any) are located in the `/api` directory
